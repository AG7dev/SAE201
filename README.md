# SAE201
Dépôt Git distant de la SAÉ 2.01 sur le développement d'une application web interactive reprenant les données conçues lors de la SAÉ 2.04 (statiques &amp; dynamiques) via les API de Data Améli. On développe une application Flask basée sur le patron de conception MVC (Modèle-Vue-Contrôleur).
